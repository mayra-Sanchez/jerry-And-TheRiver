Vidas: 3 Puntaje: 0

INSTRUCCIONES

1. Al iniciar el juego, Jerry debe comer todas las uvas y quesos que se encuentran en el mapa.

2. En las partes laterales se encuentran muros invisibles, Jerry debe evitarlos para permanecer con vida.

3. Durante el juego aparecerá el agua, que se encargará de impedir que Jerry se coma todas las uvas y los quesos.

4. Si Jerry se come todas las uvas y quesos, habrá ganado el juego.

5. Si las vidas de Jerry llegan a cero, Jerry muere y vuelve al menú.

6. Jerry se mueve con las flechas de tu teclado en cualquier dirección.
